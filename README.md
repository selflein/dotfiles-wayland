# Dotfiles for tiling window manager setup - Wayland Edition

## Symlink dotfiles to home directory
* Install GNU stow, e.g., `apt install stow`
* Navigate to the root of this git repository
* Run `stow . -t ~`

## Required packages

* `sway`
* `waybar`
* `kanshi`
* `wdisplays`
* `mako`
* `autotiling` (via python)
* `swayidle`
* `swaylock`
* `rofi`
* `alacritty`
* `slurp`
* `grim`
* `swappy`
* `clipman`
